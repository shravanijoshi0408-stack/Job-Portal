# HireNest - Job Portal

A full-stack MERN Job Portal application where recruiters can post jobs and candidates can apply online.

## Features

- User Authentication (JWT)
- Recruiter & Student Roles
- Company Management
- Job Posting & Editing
- Job Search & Filtering
- Job Applications
- Applicant Tracking
- Resume Upload (Cloudinary)
- Responsive UI

## Tech Stack

Frontend:
- React.js
- Redux Toolkit
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary
